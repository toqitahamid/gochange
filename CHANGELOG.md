# Changelog

All notable changes to GoChange are documented here.

## [1.0.0] — 2026-03-01

### Added

**iOS App**
- Home dashboard with Recovery, Sleep, and Strain score cards
- Health metrics panel: HRV, resting HR, respiratory rate, SpO2, VO2 max, steps
- Activity timeline with tap-to-view session detail
- Daily insight card with personalized recovery feedback
- Summary rings showing daily activity progress
- 4-day workout split (Push / Pull / Legs / Fullbody) with weekly progress tracking
- Active workout view: log sets with weight, reps, and RIR
- Exercise library with muscle group filters and search
- Inline previous set history during active workouts
- Progressive overload suggestions
- Minimize and resume active workouts while navigating the app
- Rest timer with Live Activity on Lock Screen and Dynamic Island
- Workout Live Activity showing set/exercise progress
- Exercise form reference via photo/video attachment
- Fitness analytics: activity heatmap, strength radar chart, cardio analytics
- Strain vs Recovery correlation chart (dual-axis, 7–30 day range)
- Session detail view for any completed workout
- Data export and import (JSON backup/restore)
- Home screen widget: weekly progress and next workout suggestion (small and medium)
- Scheduled workout reminders and rest timer notifications
- Intelligent workout scheduling based on completion history
- HealthKit write support: saves completed workouts to Apple Health

**Apple Watch App**
- Workout list view with sync from iPhone
- Active workout view (3-page: set input, overview, controls)
- Digital Crown input for weight and reps
- Real-time heart rate streaming to iPhone during workouts
- Haptic feedback on set completion
- WatchConnectivity sync for workout templates
